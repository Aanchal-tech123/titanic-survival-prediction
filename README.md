# Titanic Survival Prediction

A machine learning project that predicts whether a passenger survived the Titanic disaster.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Seaborn
- Matplotlib

## Models
- Logistic Regression
- Random Forest

## Result
Random Forest achieved approximately **85–90% accuracy**.

## Dataset
Titanic passenger dataset used for training and evaluation.
